count : 글의개수 
pageno : 페이지번호
pagesize : 10
list : 게시판 글들 
totalcount : 전체 항목 수

prev : 이전 페이지 블록의 마지막 페이지 번호 

start : 현재 페이지 블록의 시작페이지 번호

end : 현제 페이지 블록의 마지막 페이지번호 

next : 다음 페이지 블록의 첫 페이지 번호

pageno : 입력으로 받은 현재 페이지 번호 

countOfPage : 전체 페이지 수를 계산 
              전체 항목 수를 페이지 크기로 나눈 후 올림을 수행 

sizeOfBlock : 각 페이지 블록의 크기를 정의한다 
              여기서는 하드코딩된 값으로 5로 설정되어 있다 

               
prev : 이전 페이지 블록의 마지막 페이지 번호를 계산 

start : 현재 페이지 블록의 시작페이지 번호를 계산 

end : 현재 페이지 블록의 마지막 페이지 번호를 계산  만약 계산된 
      마지막 페이지 번호가 전체 페이지 수를 초과한다면 
     end 를 전체 페이지 수로 설정하고 next 를 0으로 설정
